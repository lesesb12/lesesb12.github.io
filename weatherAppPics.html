I created a web application using the OpenWeatherMap API. While developing this app, I learned how to use Javascript and jQuery to make Ajax calls to the API. After setting up my site using GitHub Pages, I was stumped on why the data from the API would not load after entering a city or zip code. I finally figured out that OpenWeatherMap does not provide a secure endpoint in their free service. Below are pictures of my weather application in action.

If you would like to check out my web application, click here. You will have to tell whichever web browser you use to load the unsafe scripts, which is designated by clicking on a shield symbol in the URL search bar.
